About these Notes

Below are my notes for first year computing modules at Imperial.

Compiling the Notes

Please note that the LaTeX notes are updated more consistently than the PDF files. The only way to ensure you get the most up-to-date set of notes is to download the plain LaTeX files and compile them yourself using pdflatex.

Errors, Corrections, Suggestions and Extensions
